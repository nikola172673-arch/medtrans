Професионален уебсайт за медицински транспорт с над 25 години опит.

## 📋 Описание

Този уебсайт е създаден за фирма Медтранс – Ивайло Лазаров ЕООД, която предоставя специализирани медицински транспортни услуги. Сайтът включва всички необходими секции и функционалности за професионално представяне на услугите.

## 🚀 Функционалности

### ✨ Основни секции
- **Херо секция** - Привлекателно представяне с основно послание

### 🎨 Дизайн и UX
- **Responsive дизайн** - Адаптивен за всички устройства (мобилни, таблет, десктоп)
- **Медицински цветове** - Професионална цветова схема в синьо и бяло

- **Интерактивни елементи** - Анимации и hover ефекти

### ⚡ Технически функционалности
- **Sticky навигация** - Менюто остава видимо при скрол
- **Smooth scrolling** - Плавно преминаване между секциите
- **Форма за валидация** - Проверка на въведените данни
- **Галерия с модал** - Пълноекранен преглед на снимки
- **SEO оптимизация** - Мета тагове и структурирани данни

## 🔧 Персонализация

### Добавяне на истински снимки
Заменете placeholder изображенията в галерията и другите секции с истински снимки:

```html
<!-- Заменете div с placeholder клас с img таг -->
<img src="images/ambulance-1.jpg" alt="Медицинска линейка" class="gallery-image">
```

### Google Maps интеграция
Добавете Google Maps в контактната секция:

```html
<iframe 
    src="https://www.google.com/maps/embed?pb=..." 
    width="100%" 
    height="250" 
    style="border:0;" 
    allowfullscreen="" 
    loading="lazy">
</iframe>
```

### Форма за имейли
Настройте формата да изпраща имейли чрез PHP или друг backend:

```php
<?php
// Примерен PHP код за обработка на формата
if ($_POST) {
    $name = $_POST['name'];
    $phone = $_POST['phone'];
    $service = $_POST['service'];
    $message = $_POST['message'];
    
    // Изпращане на имейл
    mail('ivailo_73@abv.bg', 'Ново запитване', $message);
}
?>
```
